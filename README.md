**Утилита для управления Wi-Fi через ПК без Wi-Fi адаптера. Использует Android-телефон как внешний сканер сетей.**

## Возможности
- 📡 Подключение нескольких телефонов через Wi-Fi (без USB)
- 📊 Сканирование Wi-Fi сетей с отображением уровня сигнала (RSSI)
- 🟢🟠🔴 Цветовая индикация качества сигнала
- 🔐 Детектор Evil Twin (обнаружение подозрительных сетей-двойников)
- 🗺️ Интерактивная HTML-карта с найденными сетями

## Технологии
- Python 3.13
- ADB (Android Debug Bridge)
- Folium (интерактивные карты)
- Colorama (цветной вывод в консоль)
- SubProcess (взаимодействие с CLI через код)
- random (для рандомизации некоторых действий)
- time (для взаимодействия с временем)

## Требования
- Windows
- Python 3.8+
- Android 11+ на телефоне
- ADB (Android Platform Tools)

## Установка и запуск
1. Установите Python с [python.org](https://python.org)
2. Скачайте ADB с [официального сайта](https://developer.android.com/tools/releases/platform-tools)
3. Установите библиотеки:
```bash
pip install colorama folium
```
5. Поместите adb.exe в папку `C:\adb\` или в папку с программой
6. Запустите:
```bash
python deepwebcat_scanner.py
```
